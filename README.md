# TextCompressor - Алгоритм сжатия текстовых файлов

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)
[![Version](https://img.shields.io/badge/Version-2.0-blue)](https://github.com/yourusername/TextCompressor/releases)

Программа для сжатия текстовых файлов (.txt) с графическим интерфейсом и расчетом **коэффициента Вайсманна** из сериала Silicon Valley.

## 📊 Особенности

- **Специализированное сжатие** - Оптимизировано только для текстовых файлов `.txt`
- **Гибридный алгоритм** - Комбинация LZ77 и Zlib с автоматическим выбором лучшего метода
- **Коэффициент Вайсманна** - Реализация формулы из Silicon Valley для оценки эффективности
- **Графический интерфейс** - Удобный GUI на базе Tkinter
- **Проверка целостности** - SHA-256 хеширование для гарантии сохранности данных
- **Статистика и история** - Детальная статистика всех операций сжатия

## 🚀 Установка и запуск

### Требования:
- Windows 7/8/10/11 (x64)
- Не требуется установка Python или дополнительных библиотек

### Установка:
1. Скачайте `TextCompressor.exe` из [Releases](https://github.com/yourusername/TextCompressor/releases)
2. Поместите файл в любую удобную папку
3. Дважды кликните для запуска

**Примечание:** При первом запуске программа автоматически создаст необходимые папки:
- `TextCompressorFiles/` - основная рабочая директория
- `TextCompressorFiles/Compressed/` - для сжатых файлов
- `TextCompressorFiles/Decompressed/` - для распакованных файлов

### Основные функции:

1. **Сжатие файла:**
   - Нажмите "Сжать .txt файл"
   - Выберите текстовый файл с расширением `.txt`
   - Программа автоматически выберет лучший метод сжатия
   - Просмотрите результат с коэффициентом Вайсманна

2. **Распаковка файла:**
   - Нажмите "Распаковать файл"
   - Выберите ранее сжатый файл (`.bin`)
   - Файл будет восстановлен в исходном формате

3. **Просмотр истории:**
   - Нажмите "Показать историю"
   - Увидите статистику всех операций

## 📊 Коэффициент Вайсманна

Программа вычисляет **настоящий коэффициент Вайсманна** по формуле из Silicon Valley:
W = α × (r/rb) × log(Tb) / log(T)

Где:
- `α = 2.89` (константа для Nucleus)
- `r` - наш коэффициент сжатия
- `rb` - базовый коэффициент (gzip)
- `T` - наше время сжатия
- `Tb` - базовое время (gzip)

**Исторический момент:** При достижении коэффициента **4.29** вы получаете специальное сообщение о достижении уровня Pied Piper!

## 🔧 Технические детали

### Поддерживаемые алгоритмы:
1. **Наш LZ77** - Быстрая реализация с хеш-таблицей
2. **Zlib (дефляция)** - Оптимальный уровень сжатия 6
3. **Без сжатия** - Для очень маленьких файлов (<50 байт)

### Формат сжатого файла:
┌──────────┬─────────────┬─────────────┬─────────────┐
│ Сигнатура│ Размер мета │ Метаданные │ Данные │
│ (5 байт) │ данных (4Б) │ (JSON) │ (сжатые) │
└──────────┴─────────────┴─────────────┴─────────────┘

**Сигнатура:** `SFCv2` (Smart File Compressor v2)
